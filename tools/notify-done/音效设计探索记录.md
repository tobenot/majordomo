# 🔊 notify-tone 音效设计探索记录

> 记录 `notify-tone.ps1` 提示音从初版到当前版本的设计迭代过程。  
> 所有音效均使用纯 PowerShell 逐采样合成 WAV，无任何外部依赖。

---

## 最终版本概述

**风格定位**：远方科幻警报 — 空谷回响 + 弥散混响 + 穿雾金属击打

整体画面：身处空旷的科幻场景中，远处某个巨型设施发出低沉的运转声和缓慢的警报嗡鸣，穿过浓雾和山谷传来。每一声嗡鸣的高潮处，伴随一记清晰但边缘模糊的金属敲击——这是前景提醒音，确保你能明确感知到"有通知到了"。

**时长**：约 10 秒  
**采样率**：44100 Hz / 16bit / 单声道

---

## 音频分层架构

当前版本由 4 个声音层叠加而成：

### Layer 1：远方空谷巨兽（低频底色）

| 参数 | 值 |
|------|-----|
| 频率范围 | 130~196 Hz（C3 → F3 → G3） |
| 起音方式 | 极缓慢（0.5s 平方曲线） |
| 衰减方式 | 指数衰减（`exp × -0.5`），极缓慢消散 |
| 音色 | 基频 + 五度泛音（1.5×），干净低沉 |
| 回声 | 4 层涌动式弥散回声（0.7s → 2.8s） |
| 节奏 | 3 次脉冲，间隔 2.8 秒 |

**作用**：营造空旷、苍凉的"远方巨型设施运转"底色。像隐隐约约的引擎隆隆声。

### Layer 2：缓慢警报嗡鸣（中频主体）

| 参数 | 值 |
|------|-----|
| 频率范围 | 349~440 Hz（F4 → G4 → A4） |
| 包络 | 缓起（0.6s）→ 持续 → 缓落（0.8s），总约 2.2s |
| 音色 | 基频 + 双向 chorus 失谐（±0.3%）+ 五度泛音 |
| 回声 | 5 层弥散回声（0.2s → 2.2s），逐层失谐 |
| 节奏 | 3 次脉冲，跟随 Layer 1 节奏 |
| 旋律走向 | G4 → A4（略升，渐紧迫）→ F4（下落，沉重收尾） |

**作用**：核心的"警报"辨识度。像远方防空警报穿过浓雾缓缓涌来又消退。

### Layer 2.5：穿雾金属敲击（高频前景）

| 参数 | 值 |
|------|-----|
| 频率范围 | 698~988 Hz（F5 → G5 → A5 → B5） |
| 起音 | 硬瞬态（8ms） |
| 衰减 | 中速指数衰减（`exp × -3.0`），拖着模糊尾巴 |
| 音色 | 基频 + chorus + 非谐波泛音（2.003×, 2.997×）→ 金属质感 |
| 回声 | 4 层弥散回声（0.18s → 1.4s） |
| 节奏 | 每组嗡鸣配一次"主击 + 回击"（强弱交替） |

**作用**：前景提醒音。在嗡嗡声中提供明确的"有通知到了"的感知锚点。像钟被敲了一下然后在雾中回荡。

### Layer 3：空气底噪

极微弱的伪随机噪声（振幅 0.2%），模拟环境空气感。

## 核心声学技术

### 弥散混响（Smeared Reverb）

所有声音层都使用了 4~5 层密集延迟回声来模拟巨大空间的反射：

- **涌动式包络**：回声不是"砰"一下出现，而是 `t × rate × exp(-t × rate)` 形状——缓缓涌起再散去
- **逐层失谐**：每层回声的频率会偏移 0.2%~0.4%，层次越远偏移越大，制造边界模糊的"糊"感
- **密集叠加**：多层回声的时间间隔从 0.15s 到 2.8s 不等，互相重叠形成连续的混响尾巴

### 距离感模拟（Distance Muffling）

声音在空气中长距离传播时高频被吸收，低频保留。我们通过以下方式模拟：

- 低频层只用基频 + 五度泛音，不加高次谐波
- 警报嗡鸣使用 chorus 失谐制造"厚"而非"亮"的音色
- 所有起音都是缓慢的（0.5~0.6s），没有任何尖锐的瞬态攻击（Layer 2.5 除外，它是有意为之的前景）
- 全局 3 秒平方曲线淡出

### Chorus 效果

通过叠加微失谐的正弦波（±0.3%~0.4%）制造合唱效果，让单一频率的声音听起来更宽厚、更模糊，不像电子音那样干涩。

## 迭代历程

### V1：赛博朋克三连叮

最初的版本是快速的三连击叮音，风格偏向赛博朋克 / 科幻 UI 音效。

- 短促的高频正弦波叮音（Em7 → C/E 和弦色调）
- 轻微的回声和底噪
- **问题**：太尖锐、太急促，像手机通知音

### V2：重型机械版

往更重、更机械的方向调整。

- 加入锯齿波泛音和 sub drop（120→45Hz 下坠）
- 金属碰撞的非谐波泛音
- 继电器咔嗒声瞬态
- **问题**：太近、太硬、太工业，不够"远"

### V3：远方空谷巨兽（选型 A）

用户希望"每个音间隔长、回音长、音调低、感觉从很远传来"。

进行了三个方向的选型头脑风暴：
- **A：废土灯塔 / 远方空谷巨兽** — 苍凉旷远
- **B：深水区潜航 / 巨型声纳** — 幽闭压迫
- **C：轨道空间站信标钟声** — 宏大仪式

用户选择了方向 A。

- 极长间隔（2.5s）的低频脉冲
- 多层涌动式峡谷回声
- 去掉所有高频瞬态
- **问题**：低频太重，有点伤耳朵；且只有低沉嗡鸣，缺少明确的提醒感

### V4：空谷巨兽 + 晨唤铃声

融合远方空旷感和起床铃的明确感。

- Layer 1 低频底色保留但柔化
- Layer 2 加入清脆的高频双音/三连音（C6~G6）
- 多重峡谷延迟回声
- **问题**：高频铃音太清脆利落，和模糊的背景不协调——用户说"混响不够，高频也应该糊上"

### V5：全弥散混响版

对所有声音层统一施加弥散混响处理：

- 高频铃音衰减放慢 3 倍
- 去掉高阶泛音，只留基频 + chorus
- 5 层密集回声 + 涌动包络 + 逐层失谐
- **效果**：所有声音都像隔着浓雾传来，边界模糊

### V6：弥散警报嗡鸣版

用户希望把铃声换成"慢的警报嗡鸣"。

- Layer 2 从清脆铃音改为 2.2s 缓起缓落的警报脉冲
- 中频为主（349~440Hz），双向 chorus + 五度泛音
- **问题**：全部变成嗡嗡声，没有前景音了；且音量不够响

### V7：当前版本（穿雾金属击打）

解决"不够响"和"没有前景音"两个问题：

- 总振幅从 0.82 提升到 0.95
- 各层音量提升 30~50%
- 新增 Layer 2.5 穿雾金属敲击：每次嗡鸣的高潮时刻插入一记清晰的金属击打（"铛！"），比嗡鸣高一个八度，有明确瞬态但也带弥散尾巴

## 可调参数速查

如需微调音效，以下是关键参数的位置和作用：

| 想调什么 | 改哪里 | 说明 |
|----------|--------|------|
| 总音量 | `$amplitude` 的系数（当前 0.95） | 不建议超过 0.98 以免削波 |
| 脉冲间隔 | 各层 `$beacons` / `$sirens` / `$strikes` 的 `T` 值 | 增大 T 差值 = 更慢 |
| 音高 | 各层的 `F` 值（Hz） | 降低 = 更沉，升高 = 更亮 |
| 混响"糊"度 | 回声层数和 `$Delays` 数组 | 层数越多、间隔越密 = 越糊 |
| 失谐程度 | chorus 的 `1.003` / `0.997` 等系数 | 偏差越大 = 越宽越糊 |
| 起音速度 | `$riseT` / `$att` | 越大 = 起音越慢越"远" |
| 衰减速度 | `exp` 的系数（如 `-3.0`） | 绝对值越大 = 衰减越快越干 |
| 前景击打力度 | `$strikes` 的 `Vol` 值 | 增大 = 击打更突出 |
| 淡出长度 | `$globalEnv` 中的 `3.0`（秒） | 增大 = 结尾更柔缓 |

## 技术备注

- 所有音频合成均为逐采样计算（`for` 循环遍历 44100×10 = 441000 个采样点）
- WAV 文件直接在内存中构建（`MemoryStream` + `BinaryWriter`），通过 `SoundPlayer.PlaySync()` 播放
- 无任何外部依赖，纯 PowerShell 5.1 兼容
- 采样后进行 `[-0.98, 0.98]` 硬限幅，防止削波
